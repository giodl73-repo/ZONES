# ZONES Wave

1. Read `context/waves/PHASES.md`.
2. Read the active wave `WAVE.md`.
3. Confirm the smallest pulse that advances the wave.
4. Keep source acquisition, scoring, and publication choices in ZONES.
5. Use RLINE only for reusable graph/stat/optimization primitives.
6. Update wave and pulse status.
7. Run the validation commands named in the pulse.
