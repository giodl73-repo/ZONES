# Track A - Measurement And Baseline

Papers in this track:

- A.0+solar-noon-baseline
- A.1+county-deviation-atlas
- A.2+dst-shift-analysis

## Module Question

Where are current civil time-zone assignments most misaligned with solar mean
noon, and how much of that mismatch is due to standard-zone boundaries versus
DST-period clock shifts?

## Hypothesis Gate

Before drafting, each paper must pass a `/researchprewrite`-style check:

- state falsifiable hypotheses,
- name the source data needed to test each hypothesis,
- define what would falsify the claim,
- separate standard-time, DST-period, and candidate-regime scenarios,
- identify which role panel must review the claim before publication.
