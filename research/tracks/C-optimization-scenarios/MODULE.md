Papers in this track:

## Module Question

Which candidate time-zone plans reduce solar-time error while preserving
contiguity, legal-boundary legitimacy, and minimum disruption?

## Hypothesis Gate

Before drafting, each paper must pass a `/researchprewrite`-style check:

- state the optimization objective and constraints,
- define candidate families and deterministic search procedure,
- name baseline and comparison regimes,
- define what outcome would falsify claimed improvement,
- distinguish solar-fit improvement from DOT-convenience feasibility.
